The Projects

eiseIntra

Create administrative backend for your web application, build intranet database applications with LDAP/AD integration, organize document flow with approval logic! Made with HTML5/PHP/MySQL stack.

eiseXLSX

Import and export of Excel files is easy! Read templates from your customers, fill them with data from the database and send it to user for download. Use native A1/R1C1 cell addresses. For PHP developers.

eiseMail

Need batch mail sending? Tired of PHP mail() function? Want to save sent mails to "Sent items"? Want to create mail robot? eiseMail is aimed to help with this! Read mail messages and respond from your PHP app!

docblock2web

Get your source code documented and organized with this tiny Python library that generates Markdown from PHPDocs inline documentation. JSDoc support and online version are coming soon!
